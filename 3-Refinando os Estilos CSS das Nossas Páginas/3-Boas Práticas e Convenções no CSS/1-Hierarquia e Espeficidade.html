Hierarquia do Efeito em Cascata

	No geral segue essa regra, de  qual style tem mais forca na estilização:
		
	1-CSS Inline - direto na tag ex. < h1 style="color:white;padding:30px;"> Inline < /h1>
	2-CSS Interno - style na mesma folha ex. h1{}
	3-CSS externo - style externo = ex. style.css

	Exemplo: se houver, a mesma estilização iguais para mesmo elemento nas 3 formas de style, terá mais forca a inline



Especificidade

Entendendo o calculo de especificidade:

Cálculo de Especificidade
  1. inline: 1000 pontos
  2. Id: 100 pontos
  3. Classes, pseudos-classes e atributos: 10 pontos
  4. Tag HTML: 1 Ponto
  • #exemplo h1.titulo = 111 pontos
  • a:hover = 11 pontos
  • div p = 2 pontos



!important

		Tomar cuidado ao uso excessivo !important.
		utilizar quando vincular um arquivo ou estilo externo a sua pagina, 
		assim, voce pode estiliza esse arquivo ou estilo externo sem comprometer sua estrturara.
		
		É aconselhado a estilizar a estrutura de forma natural (especificidade).
		Assim o navegador na hora de inspecionar nao precisara anular nenhum estilo duplicado.

Nivel de prioridade do efeito em cascata:

  1. Declarações com !important do usuário
  2. Declarações com !important do desenvolvedor
  3. Folha de estilos do desenvolvedor
    1. CSS inline
    2. CSS interno
    3. CSS externo
    4. Folha de estilos padrões do navegador



Herança

		.teste {      = PAI
			padding: 8px;
			border: 2px solid red;
		}
		.teste p {    = FILHO
			border: inherit;    /* inherit: herda o elemento-custumizado do pai*/
			padding: inherit;
		}
