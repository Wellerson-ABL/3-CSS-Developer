Motores de Renderização

Exemplo de motores:

		blink = para google chrome e opera
		Gecko = para Firefox
		Webkit = para iOS, Safari
		Trident - para Internet Explorer

		Cada nevegador tem um motor que renderiza a forma como cada um deve exibir o que foi programado no CSS.
		É por isso que cada navegador exibi um 'comando-tag e html' de uma forma visual diferente.

Prefixos no CSS

Cada Navegador usa um tipo de prefixo. 
     	 Para o deesenvolvedor , é importante ficar atento, porque você não sabe
 	     qual navegador o usuario vai usar.
		Então caso, alguma parte do seu site nao apareca no seu navegador, verifique qual é a 'tag' e consulte no icanuse.com
			Na sequenca adicione o prefixo. Exemplo na style da pagina.

			Prefixos:
			-moz-      Firefox
			-webkit-   Chrome/Edge/opera
			-ms-       IE(Versões antiga)
			-o-        Opera(Versões antiga)
