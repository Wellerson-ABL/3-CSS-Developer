focus

             :focus
                    Uma maneira de focus no elemento.
                    Por Exemplo: input, button ...

             :focus-within
                    Quando o elemento com :focus esta em uso, todos os demais elementos filhos e irmaos serão modificado tambem.
                    porém, o :focus-within é aplicável em outros elemento em que o :focus nao se aplica. 
                        Por Exemplo: label, h1, p ...  
           
             :focus-visible
                    Uma forma de desativar a estilizacao do elemento.
                    Exemplo:  
                    
                    .bnt-1:focus-visible {  [< VAI DESATIVAR TUDO ABAIXO]
                        outline: 3px dashed red; /*Espessura da borda do botao*/
                        outline-offset: 10px;   /*quantos em px vai ficar distante do button*/
                        background-color: palevioletred;
                        color: white;
                    }

valid e invalid, in-range e out-ofrange

            :valid  e  :invalid
                    Uma forma de aplicação de configuração para determinada codições. 
                    Quando for "Valida" vai aplicar "Ex. verde", ou  quando for  "invalida" vai aplicar "ex: red"  
                    nos formulario. 

            :in-range  e  :out-of-range]
                    São condições que vai aplica ao elemento, input por exemplo.
                    Similar a valid e invalid.

                    in-range = quando estar dentro da condição - paramentro por min e max no html.
                    out-of-range = quando esta fora da condição.

                < input type="text" type="cupom" placeholder="Apenas letras minúsculas" required>

            :optional
            Uma forma de estilizar os input que NÃO sao :required no CSS.

                Exemplo:

                border: 1px solid;
                border-radius: 5px;
                padding: 8px;
            }
