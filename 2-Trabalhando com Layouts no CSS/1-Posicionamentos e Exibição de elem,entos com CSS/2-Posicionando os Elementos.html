POSITION STICKY

		position: sticky;
		top, right, left ou bottom 0;

			O elemento fica preso ao movimento scrol do mouse na tela.
				TEm que determinar o limite [ex. top: 0;] pra ele. Exemplo:
				Elemento 13 = top:0;  | Elemento 14 = bottom:0;

Elemento 13| position: sticky;

Z-INDEX

		z-index: 1;

			O zindex determina a posição de uma camada, quando vários elementos,
			estão no mesmo lugar. 
				Cada numero representa uma camada-posição.
