FLEXBOX VS GRID LAYOUT

		PESQUISA NA DOCUMENTAÇÃO SOBRE; GRID, FLEXBOX:

			DISPLAY: FLEX; é uma forma de organizar os elementos em uma pagina dentro 
			de containers e de forma dinamica,organizandoos elementos em uma 
			dimensao: ou horizontalmente (as linhas)
			ou verticalmente (em colunas).

			Display: GRID; No CSS conseguimos trabalhar o nosso layout utilizando
			o conceito de gris. o Grid layout vai organizar os elementos das nossas 
			paginas em duas dimensões: de forma horizontal e vertical.
			Definindo linhas e colunas para que os conteudos da pagina sejam posicionados.
