Entendendo os conceitos do Grid Layout

		CSS Grid: é um recurso que permite construir layouts baseados
			em grades bidimencionais, dividindo a página em 
			regiões e definindo onde cada elemento deve ficar.



Nomenclaturas do Grid

		Grid Container:
			Elemento -pai que terá a propriedade 'display:flex' especificada.
			Ele irá 'envolver' todos os itens do grid.

		Grid item:
			São elementos que são filhos diretos do 'grid container.'

			Porém, os elementos dentro dos 'Grid item' NÃO serão considerados Grid item.
				Exemplo: Caso tenha dentro da grid item; Div, lista, imagens etc.

		Grid columns:
			As faixas verticais dos itens do grid, colunas.

		Grid rows:
			As faixas horizontais dos itens do grid, linhas.

		Grid gaps:
			São os espaços entre as linhas e colunas grid. 
		
		Grid line:
			São as divisórias que compõem a grade.
				As linhas entre as colunas são chamadas de (column lines | linhas de colunas)
				e, as que estão entre as faixas horizontais, chamadas de (row lines | linhas de linha).
		
		Grid cell:
			São as células, ou seja, cada "Quadrado" da grade, que vai entre uma grid line e outra.
				Similar aos quadrados-planilha do Excel.

Adicionando colunas e linhas ao Grid

		grid-template-columns

			Cada espaço ocupado define um número de COLUNAS do grid:
			
			grid-template-columns: espaço 1  espaço 2  espaço 3; [= a 3 Colunas]. 
			e a largura[medida] de cada [250px/200px/400px] uma das colunas. 
						
			Exemplo abaixo: 
			grid-template-columns: 250px 200px 400px;

			A ORIENTAÇÃO: É VERTICAL, MAIS A medida é na HORIZONTAL [mexe os lados.].

	
		grid-template-rows:
		
			Cada espaço ocupado define um número de LINHAS do grid:
			
			grid-template-rows: espaço 1  espaço 2; [= a 2 Linhas]. 
					e a altura[medida] de cada [50px/50px] uma das linhas. 
						
			Exemplo abaixo: 
			grid-template-rows: 50px 50px;

			A ORIENTAÇÃO: É HORIZONTAL, MAIS A medida é na VERTICAL [mexe pra cima e pra baixo].

			1fr = unidade de medida do grid, que significa fração.
			Exemplo: grid-template-columns: 1fr 1fr 2fr;
		
			Nem sempre ele terá base para definir 1fr, entao no container deve defini-se por width e height.
	
		Função repeat ()

			Uma função que repete o valor definido, no caso abaixo 20 colunas de 1fr.
				
			Exemplo: grid-template-columns: repeat(20, 1fr).
			Exemplo: grid-template-rows: repeat(20, 1fr).

			pode-se usar outras medidas, como px, %, etc.
			Exemplo: grid-template-columns: repeat(20, 100px).

		repeat(auto-fill, 300px);
			
			Ele vai preencher o espaço disponível com 300px, e se não couber mais, ele vai criar uma nova linha.
			ele vai preencher o maximo possivel na mesma linha.
			e vai manter o 'espaço alocado' para os possiveis novos items (forma invisivel). 	

		repeat(auto-fit, 300px);
			Porem, no auto-fit ele vai recolher o 'espaço Alocado', não criando novos espaços, só o necessario.
			E tambem vai preencher o espaço disponível com 300px, e se não couber mais, ele vai criar uma nova linha.

			Conclusao, a olho nu, o auto-fill e o auto-fit são iguais, porem no dev tools do chrome, eles se
			comportam de forma diferente.
			Ver no Dev Tools do Chrome, deve diminuir no zoom até ver a diferenca - os traçados.

Grid implícito e explícito

		grid-auto-flow: column; 	Faz com que os itens extras sejam alocados em colunas. 
		grid-auto-rows: 1fr;    	Define a altura das linhas automáticas.
		grid-auto-columns: 0px; 	Define a largura das colunas automáticas.
Nesse caso, toda vez que um item for adicionado, ele vai ocupar 1fr grid-auto-rows: 1fr; e na nova linha ou coluna grid-auto-flow: row; ou grid-auto-columns:column; de forma automatica.

		grid-column-end: 3; 		 Termina na coluna 3 
		grid-row-start: 1;		 Inicia na linha 1 
		grid-row-end: 3; 		 Termina na linha 3 

		Essas grids facilitam a alocação dos itens do grid.
