ENTENDENDO OS CONCEITOS DO FLEXBOX

			Flexbox conceitos:
				Permite posicionar os elementos dentro de outro elemento (container);
				Os elementos dentro do 'container' são conhecidos como 'itens'.

				display: flex; (os itens ficam lado a lado, quando dentro do mesmo container)
					Porem quando a outros container, os container ficam abaixo um do outro. 
					Exemplo: 

							O 'Container 2' tb tem Irmãos e está para 'display:flex',
							 nesse caso ficam abaixo um do outro.

				display: inline-flex; (Ficam lado a lado, os container e os items.)
					Mesmo com outros container ficará lado a lado, porem se der zoom, ele fica abaixo.
					Exemplo:

					 		A dois 'Container 1' = [Irmãos] | Eles ficam lado a lado porque esta estilizado em

PROPRIEDADE FLEX-SHRINK

			flex-shring

			Essa propriedade vai controlar o quanto o item vai diminuir, 
			caso sejanecessario.

			Se o container nao possuir espaços suficiente para 
			acomodar os elementos, eles irao diminuir pra caber.

			.item9 {  = Item 2
				background-color: yellow;
				margin: 5px;
				height: 50px;
				width: 150px;
				flex-grow: 0;
				border: 2px solid blue;
				flex-shrink: 2;
			}
				Flex-shring: 2 está diminuindo o elemento-espaço.

				Ou seja;
				* flex-shring diminui o elemento, distribuindo os espaços sobrando para os demais items .
				* flex-grow  calcula o espaço vazio no container e adiciona no elemento item no qual acomoda o codigo.

PROPRIEDADE FLEX-BASIS

			flex-basis

			A propriedade define qual o tamanho inicial dos itens do nosso flexbox, 
			antes que o espaço de seja redistribuido no  container, de acordo
			com sua dimensão.

			Esse flex sobreescreve o height e width, porem respeita [max e min-width e height]

			.item12 {  =Item 2
				background-color: yellow;
				margin: 5px;
				width: 100px;
				border: 2px solid blue;
				text-align: center;

				flex-basis: 190px;
			}

				Observe que o flex-basis faz aumentar o elemento somente.
				
				Pode-se usar tb: 
					flex-basis: min-content;  o minimo permitido.
					flex-basis: max-content;  o maximo permitido.
