Propriedade Object-Fit

	

		

		
Propriedade object-fit

		object-fit: fill; (padrao)   // Reduz a imagem para caber
		

			
			


		
		object-fit: fill; (contain) 
			       		// A imagem é redimensionada para caber dentro do elemento, mantendo a proporção da imagem original. O espaço restante do elemento será
						//	 preenchido com a cor de fundo do elemento.
		

			
				


		object-fit: fill; (cover) // A imagem é redimensionada para cobrir completamente o elemento, 
					 						//	mantendo a proporção da imagem original. A imagem pode ser cortada para caber no elemento.
		

			
			

	
		object-fit: fill; (none) // A imagem não é redimensionada. Ela mantém seu tamanho original, mesmo que isso signifique 
												// que ela não se encaixa no elemento.
		

			
			


			object-fit: fill; (scale-down) // A imagem é redimensionada para caber dentro do elemento, mas apenas se for maior que o 
														// elemento. Se a imagem for menor, ela não será redimensionada.
