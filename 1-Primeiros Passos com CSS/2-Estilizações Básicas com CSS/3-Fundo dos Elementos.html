 O código acima mostra como alterar o fundo de um elemento HTML.
		A propriedade background-color define a cor de fundo, enquanto a propriedade 
		background-image define a imagem de fundo.
		O exemplo utiliza uma imagem chamada "ACDC.jpg" localizada na pasta "img".

			}
			.exemplo8 {
				width: 300px;
				height: 300px;
				background-image: url('./img/ACDC.jpg');
				background-size: contain; /* mantém a proporção da imagem  - porém quando a imagem é pequena, ele duplica. */
				background-size: 100% 115%; /* Nesse caso, mantém a size determinada.*/
			}
