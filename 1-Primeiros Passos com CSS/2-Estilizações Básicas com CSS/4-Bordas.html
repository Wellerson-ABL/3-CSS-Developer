TAMANHO DA BORDA

	

		
		
		
bordas

		
Podemos definir as configurações das nossas bordas através das propriedades:


		
		
    
			
  • border-width
    • 
			
  • border-style
    • 
			
  • border-color
    • 
		

		
Exemplo:

		


		
		< style>
				pre {
					background-color: rgb(240, 240, 240);
					padding: 10px;
					border-radius: 5px;
					border: rgb(71, 6, 32) solid 1px;
					font-size: 11px;
				}
				body {
					font-family: roboto, sans-serif;
				}
				.exemplo {
					background-color: lightcoral;
					height: 200px;
					width: 200px;
					border-style: solid;
					border-width: 15px;
					border-width: thin;   /* borda fina = 1px */
					border-width: medium; /* borda media = 3px */
					border-width: thick;  /* borda grossa = 5px */
					border-width: 10px 20px; /* 10px parte de cima/baixo - 20px dos lados -  */
					}
		< /style>
			
			< h3>bordas< /h3>
			< p>Podemos definir as configurações das nossas bordas através das propriedades:< /p>
	
			
			< ul>
				< li>border-width< /li>
				< li>border-style< /li>
				< li>border-color< /li>
			< /ul>
			< h3>Exemplo:< /h3>
			< div class="exemplo">< /div>
