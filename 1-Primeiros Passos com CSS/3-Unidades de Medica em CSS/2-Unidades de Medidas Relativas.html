Unidade de Medida Em

	


		em duplica sempre o pai.
		quando nao hovuer o pai, por padrao o navegador vai determinar
		para 1em = 16px
			2em = 32px
					se o pai for 30px
						entao, filho= 1em = 60px


UNIDADE DE MEDIDA REM

	


		ele sempre vai duplicar o pixels padrao do navegador - elemento raiz do html = 16px
		entao ele sempre vai duplicar esse elemento.

		Entao, ele nao vai duplicar o elemento pai como base, vai sempre duplicar o elemento raiz, 
		diferente da Medida Em.

		1rem = 16
		2rem = 32
		
					se o pai for 30px
						entao, filho= 1em = 16px


DIFERENCA ENTRE REM E EM

		Rem pode multiplar o Em - e vice e versa.

UNIDADE DE MEDIDA EX e CH

	

		EX= A unidade de medidda 'ex' irá usar o tamanho 
		da Alturado caractere X(em maiúsculo) da fonte
		do elemento.


		CH= O 'ch' é uma unidade de medida que possui como referencia
