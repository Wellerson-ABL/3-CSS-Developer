	1 - < link rel="stylesheet" href="./assets/css/style.css"> = link externo do style class="css"
	2 - Criando a pasta: nessa ordem: assets\css\style.css (o nome 'style' pode ser alterado)
	3 - style.css = é tipo de css externo. E pode ser reutizádo em varias paginas.
	    Signica que pode-se criar varios tipos de style.css com diferentes fontes de estilo, e usar em diferentes paginas.
	4 - Css Inline: é quando voce aplica o style individualmente em cada tag. Exemplo:  < h5 style="color:blue"> DIO é a melhor plataforma de programação education < /h5 >  
		Essa CSS inline somente para casos individual.
