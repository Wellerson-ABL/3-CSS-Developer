Seletor por ID (#)

	
 
		

		Esse tipo de style é tag especifico, ou seja, vai afetar a tag com id.: Exemplo::
		
Títulos


		< style>
		#titulo {
			color: blue;
			background: white;
		}
		< /style>

		< h1 id="titulo"> Títulos < /h1>


Seletor por Classe (.)

	
 

		Esse tipo de style é tag especifico, ou seja, vai afetar a tag com class.: Exemplo::
		
 Bem Vindo ! 


		< style>
		.varios {
			color: blue;
			background: white;
		}
		.texto-maiusculo {
			text-transform: uppercase;
		}
		< /style>

		< h1 class="varios texto-maiusculo"> Bem Vindo ! < /h1>

		TODOS OS ELEMENTOS COM 'ESSAS CLASSES' VAO SER AFETADOS.
		Na parte Class="varios texto-maiusculo texto-italico" | podem ter varios elementos de identificados. Nesse exemplo tres identificadores: varios, texto-maiusculo, texto-italico 
		OBS: todos seletores nao permitem espaços entre o nome da classe e o ponto ou nos nomes Exemplo: .varios,.bem-vindo etc.


Seletor Universal (*)

	
 
		
		
		< style>
			* {
				font-weight: bold;
			} 
		< /style>

		NESSE CASO: Esse comando faz toda a pagina ficar em NEGRITO.
		Esse comando (*) faz aplicação universal, em todas as tags, mesmo que elas ja tenhas CSS aplicado.
